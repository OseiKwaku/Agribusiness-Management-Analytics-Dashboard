# Agribusiness-Management-Analytics-Dashboard

### Project Overview
This project presents an end-to-end agribusiness analytics solution developed using Power BI. It focuses on analyzing farm performance, productivity, and financial outcomes to support data-driven decision-making in agriculture.

The dashboard provides insights into:

- Financial performance (revenue, profit)
- Farm productivity and efficiency
- Input utilization (fertilizer, labor, irrigation)
- Market distribution and export activity
- Post-harvest storage losses

### Objectives
- Evaluate profitability across farms, crops, and regions
- Analyze drivers of productivity and efficiency
- Identify operational inefficiencies (e.g., storage loss, input usage)
= Support strategic decision-making in agribusiness

 # Dashboard Pages
### 1. Executive Overview
Provides a high-level summary of key business metrics and trends.
Key KPIs:

- Total Revenue: 625.5M
- Total Profit: 250.36M
- Average Profit per Farm: 16.43K
- Total Yield: 1.11M tons
- % Farms Using Credit: 57.83%
Key Insights:

- Strong overall financial performance across regions
- Noticeable variation in profit margins by crop type
- Balanced distribution across domestic, regional, and international markets
- Revenue trends show seasonal fluctuations

  
### 2. Farm Productivity & Efficiency
Focuses on how inputs and farm characteristics influence output and profitability.
Key Metrics:

- Yield per Hectare: 2.86
- Labor per Hectare: 54.75
- Fertilizer per Hectare: 104.94
Key Insights:

- No strong linear relationship between fertilizer use and yield, suggesting inefficiencies
- Farm size does not directly correlate with higher profit, indicating operational differences
- Variation in yield across crop types highlights opportunity for optimization
- Storage losses remain a critical issue, impacting overall profitability
- Irrigation methods show differing productivity outcomes

### Key Business Insights
- Profitability is driven more by efficiency and management practices than farm size
- Input optimization (fertilizer, labor) is essential for improving yield
- Post-harvest losses significantly reduce effective revenue
- Export markets provide diversification but vary in contribution
- Credit access is widespread but requires performance monitoring

### Tools & Technologies
- Power BI (Data modeling, DAX, dashboard design)
- Excel (Data preparation)
- DAX (Measures for KPIs, YoY growth, efficiency metrics)

  
### Dataset
A synthetic dataset was created to simulate real-world agribusiness operations.
Features include:

- Farm characteristics (size, region, crop type)
- Financial metrics (revenue, cost, profit)
- Productivity metrics (yield, labor, fertilizer)
- Logistics & storage (distance, storage loss)
- Market data (export type, quantity)
- Credit usage and interest rates
